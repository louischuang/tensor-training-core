# SKILL.md Usage For Third-Party Platforms And Agents

## Purpose

`SKILL.md` describes the supported repository workflow for AI agents and external platform integrations.

## What It Covers

- supported tasks
- supported CLI and API interfaces
- expected artifact locations
- runtime expectations
- dataset and licensing caveats

## When To Use It

Use `SKILL.md` when an external automation platform or agent needs a stable contract for:

- preparing datasets
- training models
- exporting `.tflite`
- packaging mobile assets
- reading artifacts and logs
